# 📓 OJT Daily Journal System

A modern, web-based daily journal system designed for OJT (On-the-Job Training) students to log their daily activities and for administrators to monitor progress and compliance.

## ✨ Features

- **Role-Based Access**: Separate dashboards for Students and Administrators.
- **Daily Journaling**: Students can write and save drafts or submit daily journals.
- **Progress Tracking**: Automatic tracking of submitted vs. missing journals based on the student's OJT start date and work schedule.
- **Admin Review**: Administrators can review, provide remarks, and categorize submissions.
- **Modern UI**: Clean, responsive, and premium design with glassmorphism effects.
- **Firebase Integration**: Powered by Firebase Auth and Firestore for real-time data management.

## 🚀 Quick Start (Local Setup)

Since this project uses ES Modules and Firebase, it must be run through a local web server.

1.  **Clone the repository** (or download the files).
2.  **Open a terminal** in the project root folder.
3.  **Run a local server**:
    - Using Node.js: `npx serve .`
    - Using Python: `python -m http.server 8000`
4.  **Access the app**: Default link is usually `http://localhost:3000` or `http://localhost:8000`.
